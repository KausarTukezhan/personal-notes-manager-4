Technical Documentation

GOLDEN NOTES is a secure, full-stack web application designed for personal information management. The project demonstrates a robust Client-Server architecture, implementing secure data persistence and RESTful API principles.

BACKEND

Built with Node.js and Express.js. Features modular routing, custom middleware for authentication, and centralized error handling.

DATABASE

Powered by MongoDB Atlas. Utilizes non-relational document storage for flexible and scalable data management.

SECURITY

Implements Session-based authentication with encrypted cookies and password hashing via the Bcrypt library.

FRONTEND

Developed using Vanilla JavaScript and Fetch API to ensure a fast, reactive User Interface without heavy frameworks.

Key Technical Features:

  • User Lifecycle: Secure registration, login, and session management.
  • Full CRUD Operations: Complete Create, Read, Update, and Delete logic for records.
  • Server-Side Logic: Real-time search filtering and database pagination to optimize performance.
  • Data Integration: Hybrid storage approach using both NoSQL (MongoDB) and File System (JSON).
  • Responsive Design: Modern UI crafted with CSS Grid and Flexbox for cross-device compatibility.

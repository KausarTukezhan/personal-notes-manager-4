API Test Links

Test the API endpoints directly in your browser:

Basic CRUD Operations

  • GET /api/notes - Get all notes
  • POST /api/notes - Create a new note (use API client like Postman)
  • PUT /api/notes/:id - Update a note (use API client)
  • DELETE /api/notes/:id - Delete a note (use API client)

Filtering Examples

  • GET /api/notes?category=work - Filter by category
  • GET /api/notes?search=test - Search in title and content
  • GET /api/notes?filter={"category":"personal"} - Advanced filter (JSON)

Sorting Examples

  • GET /api/notes?sort={"createdAt":-1} - Sort by date (newest first)
  • GET /api/notes?sort={"createdAt":1} - Sort by date (oldest first)
  • GET /api/notes?sort={"title":1} - Sort by title (A-Z)

Projection Examples

  • GET /api/notes?fields=title,category - Only show title and category
  • GET /api/notes?fields=title,createdAt - Only show title and date

Combined Examples

  • Work notes, sorted by date, title only
  • Search "important", sorted by title

Sample API Calls (use curl or Postman)


# Create a note
curl -X POST http://localhost:3000/api/notes \
  -H "Content-Type: application/json" \
  -d '{"title":"My Note","content":"Note content","category":"personal"}'

# Update a note (replace :id with actual ID)
curl -X PUT http://localhost:3000/api/notes/:id \
  -H "Content-Type: application/json" \
  -d '{"title":"Updated Title","content":"Updated content","category":"work"}'

# Delete a note (replace :id with actual ID)
curl -X DELETE http://localhost:3000/api/notes/:id
